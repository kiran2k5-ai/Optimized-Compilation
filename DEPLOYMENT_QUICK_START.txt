🚀 5-STEP DEPLOYMENT (RENDER.COM)

STEP 1: Sign Up (2 minutes)
────────────────────────────────────────────────────────────────────────────
Go to: https://render.com
Click: "Get Started"
Login: With GitHub account

STEP 2: Create Web Service (2 minutes)
────────────────────────────────────────────────────────────────────────────
Dashboard → "New+" → "Web Service"
Select: GitHub repo "Optimized-Compilation"
Service name: coderunner-academy
Branch: main
Click: "Create Web Service"

STEP 3: Add Database (3 minutes)
────────────────────────────────────────────────────────────────────────────
Dashboard → "New+" → "MySQL"
Name: coderunner-db
Click: "Create"
Copy connection details

STEP 4: Set Environment Variables (2 minutes)
────────────────────────────────────────────────────────────────────────────
Web Service → "Environment"

Paste these variables:
┌────────────────────────────────────────────┐
│ DB_HOST=[from MySQL service page]         │
│ DB_NAME=moodle                            │
│ DB_USER=admin                             │
│ DB_PASSWORD=[from MySQL service page]     │
│ MOODLE_URL=coderunner-academy.onrender.com│
│ CODERUNNER_ENABLED=true                   │
│ SECURITY_ENABLED=true                     │
│ DEBUG_MODE=false                          │
└────────────────────────────────────────────┘

Click: "Save"

STEP 5: Deploy & Test (10 minutes)
────────────────────────────────────────────────────────────────────────────
Render automatically starts deployment
Wait for: "Live" status
Access: https://coderunner-academy.onrender.com
Test: Login → Create quiz → Submit code

⚠️ COMMON ISSUES & FIXES

Issue: "502 Bad Gateway" Error
└─ Fix: Check Logs in Render → Service → Logs

Issue: "Database Connection Failed"
└─ Fix: Verify DB_HOST variable matches Render MySQL

Issue: "File not found" Error
└─ Fix: Git push to trigger re-deployment: git push origin main

Issue: "Connection refused"
└─ Fix: Wait 5 minutes for Render to fully start service
